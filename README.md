# fuckcdn
Fuckcdn is an alternative to CDNs
You can clone this repo to your local server to host your libs without replacing all paths (just add fuckcdn.my.site/ to every CDN load)

## CDNs

- ajax.aspnetcdn.com
- ajax.googleapis.com
- ajax.microsoft.com
- cdn.jsdelivr.net
- cdnjs.cloudflare.com
- code.jquery.com
- lib.sinaapp.com
- libs.baidu.com
- yandex.st
- yastatic.net
